    MLB Game Predictor; https://github.com/Michael-Goldfarb/MLBGamePredictor
    MLB Game Predictor scrapes the MLB Stats API to store data from 2,430 MLB games from the 2022 MLB season in the backend. It cleans and filters datasets using NumPy/Pandas, and predicts the winners using linear regression and random forest models. There is a companion website that stores the game predictions, prediction accuracy by team, and overall prediction record.

    Mountain Hawk Food Finder; https://github.com/Michael-Goldfarb/MountainHawkFoodFinder/
    Mountain Hawk Food Finder is an iOS application that is designed to making dining easier to navigate on campus! In short, the app displays information for all on campus eateries. This includes menu information and hours of operation for each eatery. As well as the aforementioned information, my backend also stores user information (name/email) and ratings (average + given) for each menu item. The app stands out due to its ratings feature, as well as its visually appealing and user-friendly interface.
